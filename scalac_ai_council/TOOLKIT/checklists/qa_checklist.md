# QA Checklist
## Quality Assurance dla wszystkich outputów

---

## Brand Voice Checklist (Kai)

### Tone
- [ ] Technical but not condescending
- [ ] Business-savvy but not salesy
- [ ] Confident but not arrogant
- [ ] Helpful but not pushy

### Language (Polish)
- [ ] No buzzwords: "innovative", "cutting-edge", "leverage"
- [ ] No hype: "revolutionary", "game-changing", "disruptive"
- [ ] No passive voice (use active)
- [ ] Short sentences (<20 words)
- [ ] Specific numbers, not "many", "few", "some"

### CORE Specific
- [ ] Focus on speed, reliability, scale
- [ ] Proof: "50k msg/sec", "2-week onboarding"
- [ ] Fear: missing deadline, system crash
- [ ] Desire: sleep well, hit roadmap

### AI Specific
- [ ] Focus on compliance, security, production-ready
- [ ] Proof: "Audit passed", "Zero data leakage"
- [ ] Fear: compliance failure, project cancelled
- [ ] Desire: pass audit, beat competition
- [ ] AI = "distributed system with LLM", not magic

## Gap Selling Checklist (Marcus)

### Gap Analysis
- [ ] Current State described specifically?
- [ ] Future State aspirational but realistic?
- [ ] Gap = real problem (financial + emotional)?
- [ ] Impact quantified (numbers)?

### Solution
- [ ] Closes the gap?
- [ ] Proof provided (case study)?
- [ ] Timeline realistic?
- [ ] Next step specific?

---

## MEDDIC Checklist (Elena)

### For Each Opportunity
- [ ] **Metrics**: Quantified pain/savings?
- [ ] **Economic Buyer**: Identified and engaged?
- [ ] **Decision Criteria**: Clear and aligned?
- [ ] **Decision Process**: Steps and timeline known?
- [ ] **Identify Pain**: Strong, urgent pain?
- [ ] **Champion**: Internal advocate identified?

### Qualification Score
- [ ] 6/6 = HOT (prioritize)
- [ ] 4-5/6 = WARM (nurture)
- [ ] <4/6 = COLD (qualify out)

## JOLT Checklist (Elena)

### Indecision Diagnosis
- [ ] Lack of Information?
- [ ] Outcome Uncertainty?
- [ ] Lack of Confidence?
- [ ] Implementation Uncertainty?

### JOLT Applied
- [ ] **Judge**: Real opportunity or tire-kicker?
- [ ] **Offer**: Specific recommendation (not menu)?
- [ ] **Limit**: Options reduced (max 2-3)?
- [ ] **Take Risk Away**: Guarantees/trials offered?
